Label Plugin Examples

No options (Has click alert to show label event pass through)

map.labels(); map.svg.selectAll('.datamaps-subunit').on('click', function(geography) {     alert('Clicked: ' + geography.id); });

Small subunits only

map.labels({     smallSubunitsOnly: true });

Small subunits only with custom list

map.labels({     smallSubunitsOnly: true,     smallSubunits: ['RI', 'CT', 'MD', 'DE'] });

Disable passing label mouse events to original subunit (click and hover won't pass)

map.labels({     passLabelMouseEvents: false }); map.svg.selectAll('.datamaps-subunit').on('click', function(geography) {     alert('Clicked: ' + geography.id); });

Label Style Options

map.labels({     fontFamily: "Verdana",     fontSize: 13,     labelColor: '#00a',     labelSpacing: 10,     labelStartCoordinates: [-65.707617, 45.722131],     lineColor: "#175635",     lineWidth: 2 });

Disable popup on label hover

map.labels({     popupOnLabelHover: false }); map.svg.selectAll('.datamaps-subunit').on('click', function(geography) {     alert('Clicked: ' + geography.id); });
